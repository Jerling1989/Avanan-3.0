How Avanan Works





Watch Video

Intelligent Scanning

  • Trained on advanced threats

    Describe your home online and get appraisal in 3 minutes. Scan emails after default security, Secure Email Gateways (SEGs), and Advanced Threat Protection. Learning from the attacks those technologies miss, Avanan blocks attacks that evade traditional scans before they reach the inbox.

  • Adapted to each environment

    Artificial Intelligence (AI) learns from relationships between employees, historical emails, and communication patterns to build a custom threat profile that blocks attacks specific to each organization.

  • Invisible to Hackers

    No change to MX records means that it is impossible for hackers to see if an organization is using Avanan to secure their cloud. At the same time, the seamless security doesn’t interrupt user experience in cloud applications.

Full-Suite Protection

  • Security Beyond Email

    Avanan connects directly to the native API of cloud email providers and their associated SaaS applications, providing real time and historical data on every user, file, event, and policy — not only of internal accounts, but everyone who has access.

  • Universal Policy Control

    Avanan maps the user, file, and permission conditions of each cloud into a single threat management interface. This enables universal policy control across different SaaS applications.

  • Instand and Secure Connection

    Connect Avanan to a new SaaS application in seconds using the latest OAuth token-based, TLS-encrypted protocols. Avanan only asks for the minimum level of required access.

Layered Security

  • Industry-leading security technologies

    Utilize innovative security technology from the industry's top vendors that offer next-generation tools to prevent tomorrow's threats. Before being added to the platform, each solution is vetted to see if it can identify threats better, faster and more reliably than others on the market.

  • Single License

    Purchase multiple layers of cloud security for the same price as just one in a datacenter appliance. Each vendor's individual licensing model has been consolidated into a single per-user, per-month subscription.

  • Deploys in one click

    Choose individual security layers, or use one of Avanan pre-made bundles. One click in the interface spins up a new instance of the pre-configured security tool.
